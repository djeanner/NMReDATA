The file replaces an existing Mnova .qs file. (keep a copy before you overwrite). On mac, copy it to:
/Applications/MestReNova.app/Contents/Resources/scripts
To call it from Mnova 11:
1) load a file with some assignement
2) use the menu "Scripts/Reports/Assignments..."
It will create the nmredata.sdf files in your home directory (as version Nov. 1st 2017)
